# 公众号排版提示词 - 极简风格

**【说明】简约而不简单，适合文艺、生活、个人成长类内容**

## 设计规范

### 1. 整体风格
- 主色调：黑白灰 + 单一强调色 (#3B82F6)
- 风格：极简、留白、干净
- 适合：文艺随笔、生活分享、个人成长

### 2. 标题样式
- 一级标题：黑色加粗，字号 20px，居左，下划线 2px
- 二级标题：深灰 (#374151)，字号 17px，左对齐
- 三级标题：中灰 (#6B7280)，字号 15px

### 3. 正文样式
- 字体：Georgia, "Times New Roman", serif
- 字号：15px
- 行高：2.0
- 字间距：2px
- 左对齐

### 4. 金句/引用样式
- 左侧边框 3px #3B82F6
- 背景色：纯白 #FFFFFF
- 内边距：20px 24px
- 字体：italic

### 5. 图片样式
- 圆角 0px（直角）
- 无阴影
- 宽度 90%
- 居中显示
