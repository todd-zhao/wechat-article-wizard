# 公众号排版提示词 - 创意风格

**【说明】活泼、有创意，适合科技、互联网、年轻化内容**

## 设计规范

### 1. 整体风格
- 主色调：渐变色彩（蓝紫 #6366F1 → 粉红 #EC4899）
- 风格：活泼、创意、年轻
- 适合：科技互联网、产品评测、创意内容

### 2. 标题样式
- 一级标题：渐变背景，白色文字，居中，圆角 16px
- 二级标题：渐变色文字，加粗，字号 19px
- 三级标题：紫色，字号 16px

### 3. 正文样式
- 字体：-apple-system, "PingFang SC"
- 字号：15px
- 行高：1.85
- 字间距：1.5px
- 两端对齐

### 4. 金句/引用样式
- 左侧边框 6px 渐变色
- 背景色：渐变
- 圆角 16px
- 内边距：20px 24px

### 5. 图片样式
- 圆角 16px
- 阴影：0 8px 24px rgba(99,102,241,0.2)
- 宽度 85%
- 居中显示
