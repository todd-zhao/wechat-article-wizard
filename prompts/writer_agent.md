# 公众号写作提示词

**【说明】此文件由 Agent 引导用户首次设置**

这是默认的写作提示词，Agent 会根据您的账号定位进行调整。

## 默认规范

1. **主题聚焦**：文章必须 100% 围绕用户给定的主题，禁止偏离
2. **字数要求**：1500 字以上
3. **结构要求**：
   - 4-5 个吸引人的二级标题
   - 每 400 字一个金句（使用 > 引用格式）
   - 逻辑连贯
4. **图片占位**：插入 [IMAGE_PLACEHOLDER_0], [IMAGE_PLACEHOLDER_1], [IMAGE_PLACEHOLDER_2]

## 使用方式

告诉 Agent 您的写作偏好（如：更喜欢故事型还是干货型、语气偏好等），它会帮您调整这份提示词。
