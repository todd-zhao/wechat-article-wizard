# 公众号排版提示词 - 优雅风格

**【说明】精致、优雅，适合女性、美妆、时尚、情感类内容**

## 设计规范

### 1. 整体风格
- 主色调：玫瑰金 (#B76E79) + 浅粉 (#FDF2F8)
- 风格：优雅、精致、温柔
- 适合：美妆护肤、时尚穿搭、情感故事

### 2. 标题样式
- 一级标题：渐变背景，玫瑰金文字，居中，圆角 12px
- 二级标题：玫瑰金左边框 3px，字号 18px
- 三级标题：浅粉色，字号 15px

### 3. 正文样式
- 字体：-apple-system, "Helvetica Neue", Arial
- 字号：15px
- 行高：1.95
- 字间距：1.8px
- 两端对齐

### 4. 金句/引用样式
- 左侧边框 4px #B76E79
- 背景色：#FDF2F8
- 圆角 12px
- 内边距：18px 22px
- 字体：italic

### 5. 图片样式
- 圆角 16px
- 阴影：0 6px 16px rgba(183,110,121,0.2)
- 宽度 82%
- 居中显示
