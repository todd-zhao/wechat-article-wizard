# 账号定位策略

**【说明】此文件由 Agent 引导用户首次设置**

首次使用时，请让 Agent 根据以下问题帮助您明确账号定位：

1. **目标读者**：您的文章面向哪些人群？（如：企业主、职场人士、宝妈等）

2. **核心价值**：您能提供什么独特价值？（如：AI 工具推荐、育儿经验分享等）

3. **账号人格**：您希望以什么形象示人？（如：专业顾问、贴心朋友、犀利评论者等）

4. **写作风格**：
   - 语气：（如：轻松幽默、专业严肃、犀利直白等）
   - 结构：（如：总分总、故事型、清单型等）
   - 特色：（如：金句频出、案例丰富、干货满满等）

5. **内容主题**：您主要写哪些方面的内容？

---

**使用方式**：告诉 Agent 您的以上信息，它会帮您填充这份定位文档。
